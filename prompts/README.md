# Book Explorer - Learning Prompts

This directory contains structured prompts for building and documenting the Book Explorer application as part of a Gen AI class project.

## Overview

These prompts guide you through building a complete Next.js application from scratch, with each phase focusing on specific concepts and features.

## Prompt Files

### 1. **ai-prompts.txt**
- General Next.js learning prompts for React developers
- Core concepts and mental models
- Advanced patterns and integration
- Project-based learning exercises
- Common issues and troubleshooting
- Supplementary prompts for understanding Next.js fundamentals

## Learning Phases

The prompts are organized into phases covering:

### Phase 1: Orientation & Bridging from React
- Structured learning plan
- Philosophical shift from React SPA to Next.js
- Practical code differences
- Tooling and setup

### Phase 2: Core Concepts Deep Dive
- Server-Side Rendering (SSR) vs Client-Side Rendering (CSR)
- Static Site Generation (SSG)
- Incremental Static Regeneration (ISR)
- Debugging strategies

### Phase 3: Advanced Patterns & Integration
- Nested layouts
- Dynamic routes
- Protected routes
- State management (server vs client)
- API Route Handlers
- Performance optimizations

### Phase 4: Project-Based Learning & Troubleshooting
- Mini-project exercises
- Debugging techniques
- Deployment to Vercel
- Backend API integration
- Testing strategies

## Common Issues Covered

1. **Params Promise Error (Next.js 16+)**
   - Dynamic route parameter handling

2. **Hydration Mismatch Warning**
   - Browser extension conflicts

3. **Module Path Alias Not Working**
   - jsconfig.json configuration

4. **Image Domain Configuration**
   - External image sources

5. **API Fetch Errors**
   - Error handling and loading states

6. **TypeScript Definition Errors**
   - JavaScript project configuration

## Quick Start

```bash
# Navigate to project
cd book_explorer

# Start development server
npm run dev

# Read the prompts
cat prompts/ai-prompts.txt
```

## Learning Outcomes

After working through these prompts, you will:
- Understand Next.js App Router fundamentals
- Master server and client components
- Implement data fetching strategies
- Handle routing and navigation
- Manage state effectively
- Deploy production-ready applications
- Troubleshoot common Next.js issues
